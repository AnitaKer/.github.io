### 🔬 What I'm Currently Working On

I am currently working on my Master's thesis: **Development of a Screening Model for Cognitive Performance (CPM) in Kilifi, Kenya.**
*   **The Goal:** Building and internally validating a harmonized machine learning screening model to predict cognitive performance in children (aged 6-11).
*   **The Challenge:** Harmonizing secondary data from three distinct, independent pediatric cohorts (NDD, NEMO, NWO) using a Common Data Model (CDM).
*   **The Methods:** Data forensics & missingness mapping, elastic net/LASSO regression, and generating age-adjusted percentile norms.
